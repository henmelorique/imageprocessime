Processamento de Imagens

Plataforma gratuita para o processamento de imagens

Sobre

A plataforma Processamento de Imagens faz parte de um dos Projetos de Fim de Curso desenvolvidos na Seção de Engenharia Cartográfica (SE/6) do Instituto Militar de Engenharia no ano de 2019. Os autores são Henrique Silva Melo e William Luna Batista, sob orientação do Coronel José Wilson Cavalcante Parente Júnior.

O projeto faz parte do portifólio que busca desenvolver ferramentas de processamento de imagens empregando a linguagem de programação Python, com propósitos principalmente acadêmicos. Uma das premissas do projeto foi que ele tivesse código aberto, de modo a poder ser editado mesmo depois da graduação de seus autores originais, permitindo melhorias constantes no sistema.
